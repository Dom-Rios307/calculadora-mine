# Calculadora Mine

Este projeto contém um script em Python e um script em Shell.

## Arquivos do projeto

calculadora.py -> Script da calculadora em Python
menu_fazer_menu.sh -> Script em Shell para executar o programa

## Como executar o arquivo .sh

1. Abra o terminal
2. Entre na pasta do projeto
3. Dê permissão de execução ao arquvo

chmod +x menu_fazer_menu.sh

4. Execute o script
./menu_fazer_menu.sh
